# 🎓 Student Profile Management System

This is a simple web-based Student Profile Management System built using HTML, CSS, and JavaScript. The system allows users to add, display, filter, search, and manage student data, including GPA and photos. It was developed as part of a group assignment in the **Software Engineering** course at Al-Qalam University, Katsina.

## 🚀 Features

- Add new student profiles with details and photo
- Display a dynamic table of students
- GPA calculation and sorting
- Search by student name
- Filter by department
- Mobile-responsive design
- Form validation
- Data saved to JSON
- Fully tested and cleaned for bugs

---

## 🧠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON (Simulated Backend)

## 📌 How to Run the Project Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/student-profile-system.git
   cd student-profile-system
````

2. Open `index.html` in your browser to view the app.

> *No server required — this project runs purely on the client side.*
